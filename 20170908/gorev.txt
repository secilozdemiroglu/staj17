- JSON nedir?

	Json (Javascript Object Notation), Javascript uygulamaları için oluşturulmuş bir veri formatıdır.

- Web servis nedir?

	Web servisleri farklı uygulamaların ağ üzerinden haberleşmesini sağlayan ve iletişimini standart bir XML 
	mesajlaşma sistemi kullanarak gerçekleştiren bir yazılım parçasıdır. XML veya json, bir web servisindeki tüm iletişimleri
	kodlamak için kullanılır. 

- Web servis client nedir?

	Web servisine istekte bulunan yapıdır.

- SOAP ve REST nedir?

	SOAP (Simple Object Access Protocol), uygulamalar ile web servislerin bilgi aktarımını sağlayan XML tabanlı bir
	protokoldür.
	
	REST(REpresentational State Transfer), Client-Server arasında veri alış verişinin basit bir yoludur.REST ile 
	veri alış verişi JSON, XML hatta Text formatında yapılabilir.

- WSDL nedir?

	WSDL (Web Services Description Language), XML fromatında yazılmış ,Web Service'lerin yerlerini, ne iş yaptıklarını
	yani metodlarını belirleyen bir dildir.
